# 🦯 โครงงานไม้เท้าอัจฉริยะเพื่อผู้พิการทางสายตา (Smart Cane)

ระบบช่วยเหลือและแจ้งเตือนอุบัติเหตุสำหรับผู้พิการทางสายตาด้วย ESP32 และ GPS

## 📖 เกี่ยวกับโครงงาน

โครงงานนี้เป็นอุปกรณ์ช่วยเหลือผู้พิการทางสายตา เพื่อเพิ่มความปลอดภัยในการเดินทาง โดยใช้บอร์ด **ESP32** ร่วมกับเซนเซอร์ต่างๆ ในการทำงาน 2 ระบบหลัก:
1. **ระบบเตือนสิ่งกีดขวาง:** ตรวจจับระยะทางด้วย Ultrasonic และแจ้งเตือนผู้ใช้ผ่านการสั่นและเสียง
2. **ระบบเตือนการล้ม:** ตรวจจับการล้มด้วย MPU6050 หากเกิดอุบัติเหตุจะส่งพิกัด GPS แจ้งเตือนไปยัง Telegram ของผู้ดูแลทันที

---

## 🛠️ อุปกรณ์ที่ต้องใช้ (Hardware)

1. บอร์ด ESP32 (Dev Kit V1)
2. เซนเซอร์อัลตราโซนิก (HC-SR04)
3. เซนเซอร์วัดการเอียงและแรงจี (MPU6050)
4. โมดูล GPS (u-blox NEO-6M)
5. มอเตอร์สั่น (Vibration Motor) และ ลำโพง (Buzzer)
6. แหล่งจ่ายไฟ (Power Bank 5V)
7. ท่อ PVC สำหรับทำโครงสร้างไม้เท้า

## 🔌 การต่อวงจร (Wiring Diagram)

### 1. เชื่อมต่อเซนเซอร์ MPU6050 (I2C)
| ขาอุปกรณ์ | ขา ESP32 | หมายเหตุ |
| :--- | :--- | :--- |
| VCC | 3.3V | ไฟเลี้ยง |
| GND | GND | กราวด์ |
| SDA | GPIO 21 | ข้อมูล I2C |
| SCL | GPIO 22 | นาฬิกา I2C |

### 2. เชื่อมต่อ GPS (NEO-6M)
| ขาอุปกรณ์ | ขา ESP32 | หมายเหตุ |
| :--- | :--- | :--- |
| VCC | 3.3V | ไฟเลี้ยง |
| GND | GND | กราวด์ |
| TX | GPIO 16 | ส่งข้อมูลเข้า ESP32 |
| RX | GPIO 17 | รับข้อมูล |

## 🔗 1. ส่วนของการเขียนโปรแกรม (Arduino IDE)

อัปโหลดโค้ดลงในบอร์ด ESP32 โดยต้องติดตั้งไลบรารีเสริมดังนี้:

* `TinyGPS++` (จัดการพิกัดดาวเทียม)
* `Adafruit_MPU6050` (จัดการเซนเซอร์การล้ม)
* `UniversalTelegramBot` (สำหรับส่งข้อความแจ้งเตือน)

## 🚀 2. วิธีการใช้งาน

**ขั้นตอนที่ 1: การเตรียมพร้อม**
1. เสียบสาย USB ระหว่างไม้เท้ากับ Power Bank
2. รอสัญญาณ GPS จับพิกัด (ประมาณ 3-5 นาทีในที่โล่ง)
3. ระบบพร้อมใช้งานเมื่อได้ยินเสียงสัญญาณเตือนสถานะพร้อม

**ขั้นตอนที่ 2: การทดสอบระบบแจ้งเตือน**
1. จำลองการล้มโดยการปล่อยไม้เท้ากระแทกพื้น
2. รอรับข้อความแจ้งเตือนฉุกเฉินพร้อมลิงก์ Google Maps ทางแอปพลิเคชัน Telegram ของผู้ดูแล
